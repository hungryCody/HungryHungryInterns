Intern Survival Guide

This tool was built to help associates and any other people that are willing to use this tool. It has a few different components, including a fun capital one themed game in the the leisure lobby, a tool to help track any available leftovers throughout capital one, and a list of helpful resources and tools to help answer quetsions that new interns may have.

The idea for this tool initially started as a project to help event holders alert anyone to leftover food for events that they would people to come and eat instead of it going to waste. We started working on it as an android app for a learning experience, but changed directions, and added a few more components to this tool.

We really wanted to learn something new while working on this project, so we decided to work with firebase. We ran into a lot of roadblocks, but it proved to be a good learning experience learning how to do development work with firebase.

We hope to expand this project, and make it more valuable and useful to interns within capital one. We hope to find useful links and information, while also building a more fun and userful tools.
